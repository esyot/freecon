About FREECON

FREECON is an open-source icon library built by developers, for developers. Created and led by Reinhard Esteban, the project empowers you to build fast, beautiful UIs with clean, scalable, and fully customizable icons.

How to Use

Using FREECON icons is easy! Simply include the CDN in your HTML and use the appropriate class names. Here's a quick example:

index.html 
<!-- Include in your <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/esyot/freecon@1731c2a/freecons.css">

<!-- Example usage in your HTML -->
<i class="freecon brand-twitter-default icon-2x icon-hover"></i>

You can also customize the size, color, and shape using classes like icon-2x, icon-blue-500, and icon-rounded-full.
