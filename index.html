FREECON

Free & Customizable Icons for Developers

About FREECON

FREECON is a powerful and flexible open-source icon library designed with developers in mind. Built by developers, for developers, it offers a growing collection of clean, modern, and fully scalable icons that can be used freely in both personal and commercial projects. FREECON was developed with the goal of making beautiful, performant user interfaces more accessible to everyone. Whether you're building a web app, mobile app, or design prototype, FREECON provides the tools you need to create consistent and visually appealing experiences—without worrying about licensing restrictions or bloated design assets.

Features

Free & Open Source

No fees. No licenses. Use in personal or commercial projects without restrictions.

Fully Customizable

Change color, size, roundness, etc. Total control through classes.

Performance Optimized

Lightweight, clean SVGs designed for fast loading and rendering.

Framework Agnostic

Use with any framework — React, Vue, Angular, Svelte, or plain HTML.

How to Use

Using FREECON icons is easy! Simply include the CDN in your HTML and use the appropriate class names. Here's a quick example:

index.html 
<!-- Include in your <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/esyot/freecon@1731c2a/freecons.css">

<!-- Example usage in your HTML -->
<i class="freecon brand-twitter-default icon-2x icon-blue-600 icon-rounded-full icon-hover"></i>

You can customize icons with utility classes:

  • Size: icon-sm, icon-md, icon-lg, icon-2x, icon-3x
  • Color: icon-blue-600, icon-red-500, icon-gray-400, etc.
  • Shape: icon-rounded, icon-rounded-full, icon-rounded-md, etc.
  • Effects: Use icon-hover for hover interactivity.
